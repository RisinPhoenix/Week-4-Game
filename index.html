Wins: 0

Instructions:

The computer will randomly pick a number on the start of the game and display it in the JumboTron.
You start with a value of zero and the crystals are all assigned random values between one and twelve. Everytime you click on a crystal it adds it's value to your value. Use the crystals to create the value equal too the computers number.
